The Ecosystem

Đội ngũ kỹ sư trong túi bạn

Mỗi lệnh `/` kích hoạt một chuyên gia AI chuyên biệt, giải quyết triệt để từng khâu.

01
/plan The Architect

Bạn chỉ cần Vibe. AI lo Logic.

Bạn không biết thiết kế Database? Không biết User Flow? Chỉ cần nói: "Làm app bán hàng giống Shopee".

  • ✓ Tự động phỏng vấn để làm rõ nghiệp vụ (Auth, Payment, Email).
  • ✓ Tự thiết kế Database Schema (PostgreSQL/SQLite) chuẩn chỉnh.
  • ✓ Tự phát hiện yêu cầu ẩn (Scheduled Tasks, Charts, Maps).
> Generating specs/checkout_flow.md...
> Detected missing requirement: 'Inventory Check'
> Added DB migration: 'CREATE TABLE orders...'
02
/visualize Creative Director

Từ ngữ thành Giao diện.

Không cần biết CSS hay Figma. Chỉ cần nói: "Giao diện tối, kiểu Cyberpunk, nhiều Neon".

  • ✓ Tự vẽ Mockup rồi code pixel-perfect bằng TailwindCSS.
  • ✓ Tự xử lý Mobile Responsive và Dark Mode.
  • ✓ Tự thêm Loading State, Error State (những thứ newbie hay quên).
03
/deploy DevOps Master

Đưa lên Internet trong 1 nốt nhạc.

Bạn không biết VPS, SSL, DNS? Workflow này sẽ cầm tay chỉ việc.

  • ✓ Tự động check Build lỗi trước khi deploy.
  • ✓ Hướng dẫn setup Domain, HTTPS miễn phí.
  • ✓ Tự động setup SEO (Sitemap, Robots.txt) và Analytics.
04
/save-brain The Librarian

AI không bao giờ "Quên não".

Vũ khí bí mật chống lại căn bệnh Context Drift của LLM.

  • ✓ Tự động quét code thay đổi và update tài liệu Docs.
  • ✓ Tự động tạo API Documentation.
  • ✓ Ngày mai quay lại chỉ cần gõ /recap là nhớ lại hết.
