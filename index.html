Install once. Use everywhere. Cài đặt 1 lần. Dùng ở mọi dự án.

# 2. PowerShell:
iex (irm https://raw.githubusercontent.com/TUAN130294/awf/main/install.ps1)
# 1. Open Terminal
# 2. Bash/Zsh:

Existing Project? Dự án đã có code?

AWF works with any codebase. Here's how to apply it. AWF hoạt động với mọi codebase. Đây là cách áp dụng.

1

Open Project in Antigravity/Cursor Mở dự án trong Antigravity/Cursor

Open your existing project folder. AWF is already globally installed - no per-project setup needed. Mở thư mục dự án của bạn. AWF đã được cài global - không cần setup cho từng project.

2

Run /recap First Chạy /recap trước tiên

Type /recap - AI will scan your codebase and summarize the architecture, tech stack, and current state. Gõ /recap - AI sẽ quét codebase và tóm tắt kiến trúc, tech stack, và trạng thái hiện tại.

3

Run /save-brain Chạy /save-brain

Type /save-brain - AI will create documentation files in docs/ folder to remember your project forever. Gõ /save-brain - AI sẽ tạo file tài liệu trong thư mục docs/ để nhớ dự án mãi mãi.

4

Start Coding! Bắt đầu code!

Now use any workflow: /plan for new features, /debug for bugs, /audit for security check. Giờ thì dùng workflow bất kỳ: /plan cho tính năng mới, /debug cho lỗi, /audit cho bảo mật.

💡

Pro Tip: Let AI Learn Your Codebase Mẹo: Để AI học codebase của bạn

After /save-brain, AI will read your docs automatically in future sessions. It's like giving AI a "memory" of your project. Run /recap every morning to restore context. Sau khi /save-brain, AI sẽ tự động đọc docs của bạn trong các phiên sau. Giống như cho AI một "bộ nhớ" về dự án. Chạy /recap mỗi sáng để khôi phục ngữ cảnh.

Uninstall Gỡ cài đặt

Need to remove AWF? Here's how. Cần gỡ AWF? Đây là cách.

Windows (PowerShell)

# Remove AWF workflows
Remove-Item -Recurse -Force "$env:USERPROFILE\.gemini\antigravity\global_workflows"

Mac / Linux (Bash)

# Remove AWF workflows
rm -rf ~/.gemini/antigravity/global_workflows

Note: This only removes AWF workflows. Your project code and docs are not affected. Lưu ý: Chỉ xóa workflows AWF. Code và docs của dự án không bị ảnh hưởng.
