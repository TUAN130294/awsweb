Bạn mệt mỏi vì AI code xong chạy không được? Hay quên trước quên sau?
AWF là bộ quy trình (workflow) giúp AI làm việc như một kỹ sư thực thụ:
Biết lập kế hoạch, biết kiểm tra lỗi, và không bao giờ quên.

Bộ công cụ của bạn

Chỉ cần gõ lệnh tương ứng vào ô chat, không cần nhớ kỹ thuật phức tạp.

/plan Quan trọng nhất

Lên Kế Hoạch

Đừng để AI code ngay. Dùng lệnh này để AI suy nghĩ, viết ra bản thiết kế chi tiết trước.

/code Thực thi

Viết Code

AI sẽ viết code dựa trên bản kế hoạch. Tự động kiểm tra lỗi và bảo mật.

/visualize

Thiết Kế Giao Diện

Bạn có ý tưởng nhưng không biết vẽ? AI sẽ giúp bạn tạo ra giao diện đẹp mắt.

/debug

Sửa Lỗi

Khi app bị lỗi, đừng hoảng. Lệnh này biến AI thành thám tử tìm ra nguyên nhân.

/refactor

Dọn Dẹp Code

Code chạy được nhưng nhìn rối mắt? Lệnh này giúp sắp xếp lại code cho gọn gàng.

/rollback

Hoàn Tác

Lỡ tay làm hỏng code? Quay lại phiên bản cũ an toàn ngay lập tức.

/save-brain

Lưu Bộ Nhớ

Cuối ngày, dùng lệnh này để "Save Game". Mai mở lại AI vẫn nhớ bạn đang làm gì.

/next

Làm Gì Tiếp?

Bị bí không biết làm gì? AI sẽ xem xét tình hình và gợi ý bước tiếp theo.

/awf-update

Cập Nhật

Cập nhật AWF lên phiên bản mới nhất.

Cài đặt 1 lần dùng mãi mãi

Mở phần Chat hoặc Terminal trong IDE của bạn và dán lệnh này vào.

Dành cho Windows (PowerShell)
iex "& { $(irm https://raw.githubusercontent.com/TUAN130294/awf/main/install.ps1) }"
Dành cho Mac / Linux
curl -fsSL https://raw.githubusercontent.com/TUAN130294/awf/main/install.sh | sh

Sau khi cài đặt, khởi động lại IDE của bạn để bắt đầu sử dụng.
