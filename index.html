Quy trình chuẩn cho AI Coding Standard Workflow for AI Coding

Đừng để AI làm việc ngẫu hứng. Hãy ép nó tuân thủ quy trình 4 bước của kỹ sư chuyên nghiệp. Don't let AI handle code randomly. Enforce a professional 4-step engineering process.

1

Giai đoạn 1: Lập Kế Hoạch Phase 1: Planning

Không được viết code ngay! AI phải hiểu và có bản vẽ kỹ thuật trước. Do not code yet! AI must understand and draft specs first.

/init DỰ ÁN MỚI NEW PROJECT

Thiết lập cấu trúc thư mục, Git, và cài đặt môi trường ban đầu. Setup folder structure, Git, and initial environment.

/recap MỖI SÁNG DAILY START

Đọc lại file `.brain` để khôi phục trí nhớ dự án cho AI. Read `.brain` file to restore project context for AI.

/plan QUAN TRỌNG NHẤT CRITICAL

AI phân tích yêu cầu, thiết kế Database, API và viết tài liệu Spec. AI analyzes requirements, designs DB/API, and writes Specs.

/next KHI BỊ BÍ STUCK?

AI tự phân tích tình hình và chỉ cho bạn bước tiếp theo cần làm gì. AI analyzes status and guides you to the next step.

2

Giai đoạn 2: Xây Dựng Phase 2: Construction

Viết code an toàn, đẹp mắt và trơn tru. Write safe, clean, and functional code.

/visualize THIẾT KẾ DESIGNER

AI vẽ giao diện (UI) đẹp mắt, hiện đại trước khi gắn logic. Generate modern UI layouts before adding logic.

/code LẬP TRÌNH CODER

Viết code dựa trên bản Plan. Tự động kiểm tra bảo mật và lỗi. Write code based on Specs. Auto-check security and bugs.

/run CHẠY APP OPERATOR

Chạy ứng dụng (npm run dev, docker up...) an toàn. Run the application safely.

3

Giai đoạn 3: Vận Hành & Sửa Lỗi Phase 3: Operations & Debug

Đảm bảo ứng dụng chạy ổn định, sạch sẽ. Ensure application stability and cleanliness.

/debug THÁM TỬ DETECTIVE

Tìm nguyên nhân gốc rễ (Root Cause) thay vì sửa mò. Find Root Cause instead of blind fixing.

/rollback HOÀN TÁC UNDO

Hoàn tác code về trạng thái ổn định trước đó nếu lỡ tay làm hỏng. Revert code to safe state if something breaks.

/refactor DỌN DẸP CLEAN UP

Tối ưu hóa, làm gọn code, xóa code thừa. Optimize and clean code, remove redundancy.

/audit BẢO VỆ SECURITY

Quét toàn bộ dự án để tìm lỗ hổng bảo mật và vấn đề tiềm ẩn. Scan project for security risks and issues.

/deploy SHIP HÀNG SHIP IT

Đóng gói Docker và đưa ứng dụng lên môi trường Production. Dockerize and deploy to Production.

4

Giai đoạn 4: Quản Trị & Tiện Ích Phase 4: Admin & Utils

Các công cụ hỗ trợ quản lý bộ nhớ và hệ thống. Tools for memory and system management.

/save-brain LƯU GAME SAVE GAME

Lưu toàn bộ kiến thức dự án vào `.brain`. Bắt buộc dùng cuối ngày. Save all project context to `.brain`. Mandatory at end of day.

/cloudflare-tunnel MẠNG NETWORK

Quản lý kết nối Tunnel để public web ra ngoài Internet an toàn. Manage Cloudflare Tunnels for secure public access.

/test KIỂM TRA TESTING

Chạy các bài kiểm thử (Unit Test, Integration Test). Run Unit and Integration Tests.

/awf-update CẬP NHẬT UPDATE

Kiểm tra và cập nhật AWF lên bản mới nhất. Check and update AWF to latest version.
