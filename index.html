⛔ The Reality (Thực tế)

Context Drift

AI forgets architecture after 15+ messages.

AI bắt đầu "quên" kiến trúc dự án sau khoảng 15 tin nhắn.

Lazy Coding

Skipping error handling, hard-coding secrets.

Code ẩu, thiếu try-catch, lộ API Key vì lười check.

Loop of Death

Fixing one bug creates two new ones.

Sửa lỗi này đẻ ra lỗi khác. Vòng lặp debug vô tận.

✅ The Engineered Way (Cách AWF)

External Memory

Forces AI to read .brain file before coding.

AI buộc phải đọc file ký ức .brain trước khi viết code.

Security Gates

/code workflow enforces security checks & tests.

Tự động kiểm tra bảo mật và viết test trước khi trả code.

Plan First

Stop AI from coding until a Spec is approved.

Cấm code vội. Phải lên kế hoạch (Spec) và được duyệt mới làm.
