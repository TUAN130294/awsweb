Hướng dẫn sử dụng với Antigravity

AWF (Antigravity Workflow Framework) được thiết kế đặc biệt cho AI Agent "Antigravity" của Google DeepMind.

🤖 Antigravity là gì?

Antigravity là AI Agent tiên tiến của Google DeepMind, được tích hợp sẵn trong các IDE như Cursor và Windsurf. Nó có khả năng:

  • ✓ Đọc và hiểu toàn bộ codebase
  • ✓ Chạy terminal commands
  • ✓ Tạo và chỉnh sửa files
  • ✓ Sinh hình ảnh (UI Mockups)
  • ✓ Lưu trữ "Bộ nhớ" (Knowledge Items)

🎯 Tại sao cần AWF?

Antigravity rất mạnh, nhưng nó cần "hướng dẫn" để biết phải làm gì. AWF cung cấp bộ 14 workflows chi tiết để:

  • → Chống Context Drift: AI không quên kiến trúc dự án
  • → Tự phát hiện điểm mù: AI hỏi về Security, SEO, Auth...
  • → Quy trình chuẩn: Từ ý tưởng đến production
  • → Ngôn ngữ Việt: Giao tiếp bằng tiếng Việt 100%

📖 Cách sử dụng AWF với Antigravity (3 bước)

1

Cài đặt AWF

Chạy lệnh cài đặt trong Terminal của dự án. AWF sẽ tải 14 file workflow vào thư mục .agent/workflows/

2

Mở Chat với Antigravity

Trong Cursor/Windsurf, mở panel Chat (thường là bên phải màn hình hoặc Cmd+L)

3

Gõ lệnh /

Gõ /plan, /code, /debug... Antigravity sẽ tự động đọc workflow và làm theo!

💬 Ví dụ hội thoại thực tế

You: /plan Làm app quản lý chi tiêu cá nhân
Antigravity: Tuyệt vời! Em sẽ giúp anh thiết kế app quản lý chi tiêu. Trước tiên em cần hỏi một vài câu:
Antigravity: 1. Có cần đăng nhập không? (Để sync dữ liệu giữa các thiết bị)
2. Có theo dõi theo danh mục không? (Ăn uống, Đi lại, Giải trí...)
3. Có cần biểu đồ thống kê không?
You: Có hết
Antigravity: OK! Em đã tạo file Spec chi tiết tại docs/specs/expense_tracker_spec.md. Anh muốn xem UI trước (/visualize) hay code luôn (/code)?

🌟 Khởi động & Ngữ cảnh

/init Project Bootstrapper

Tạo dự án mới từ số 0. AI tự chọn tech stack, setup Git, Env, Code quality tools.

💡 Khi nào dùng: Bắt đầu project hoàn toàn mới

/recap Memory Retriever

Tóm tắt dự án khi quay lại sau nhiều ngày. AI đọc docs, git log và nhắc việc cần làm.

💡 Khi nào dùng: Sáng hôm sau, sau kỳ nghỉ

/save-brain The Librarian

Lưu kiến thức vào docs. AI tự update Architecture, API Docs, Changelog để không bị quên.

💡 Khi nào dùng: Cuối buổi làm việc, sau feature lớn

🎯 Phát triển tính năng

/plan The Architect

Thiết kế tính năng toàn diện. AI tự hỏi về DB, Auth, Payment, Charts, Maps và viết Spec chi tiết.

💡 Điểm mù xử lý: 15+ (Database, Auth, Scheduled Tasks...)

/visualize Creative Director

Thiết kế UI từ "Vibe". AI vẽ mockup, code TailwindCSS, xử lý Mobile, Dark Mode, Accessibility.

💡 Điểm mù xử lý: Loading State, Error State, Empty State

/code Senior Developer

Viết code chất lượng. AI tự thêm Input Validation, Error Handling, Security (SQL Injection, XSS).

💡 Điểm mù xử lý: Bảo mật, Performance, Logging

⚙️ Vận hành

How To Use

Hướng dẫn sử dụng từng bước

Đây là quy trình chuẩn để biến ý tưởng thành sản phẩm hoàn chỉnh.

1
/init Tạo dự án mới

Mở Terminal trong thư mục rỗng, gõ /init. AI sẽ hỏi bạn muốn làm Web App, API hay Mobile. Sau đó tự động dựng khung dự án, cài dependencies, setup Git.

User: /init
AI: "Anh muốn làm loại app gì? A) Website B) Web App C) API Backend..."
2
/plan Mô tả ý tưởng

Gõ /plan rồi mô tả ý tưởng bằng tiếng Việt bình thường. VD: "Làm app bán hàng online có giỏ hàng, thanh toán VNPay". AI sẽ tự động phỏng vấn bạn về các yêu cầu ẩn (Auth, Email, Backup...), rồi viết file Spec chi tiết.

User: /plan Làm app quản lý kho hàng
AI: "Anh có cần đăng nhập không? Có mấy loại người dùng?..."
AI: "Đã tạo file docs/specs/inventory_spec.md"
3
/visualize Thiết kế giao diện

Gõ /visualize và mô tả "vibe" của giao diện. VD: "Giao diện tối, hiện đại, kiểu Apple". AI sẽ vẽ mockup, xin duyệt, rồi code HTML/CSS giống hệt.

User: /visualize Dashboard quản lý, vibe sáng sủa chuyên nghiệp
AI: "Generating mockup..." (hiển thị hình ảnh)
AI: "Anh thấy OK chưa? Em code luôn nhé?"
4
/code Viết code

Gõ /code và AI sẽ đọc Spec, rồi triển khai code. AI tự động thêm Security (validation, sanitization), Error Handling mà bạn không cần nhắc.

User: /code
AI: "Em đọc được Spec inventory_spec.md. Bắt đầu code..."
AI: "Đã tạo: src/services/inventory.ts, src/api/inventory.ts..."
5
/run Chạy thử

Gõ /run. AI tự detect môi trường và chạy lệnh phù hợp (npm run dev, docker-compose up...). Sau đó báo URL để bạn mở browser xem kết quả.

6
/test Kiểm tra

Gõ /test để chạy unit test. Nếu chưa có test, AI tự tạo một script verify logic cơ bản.

7
/deploy Đưa lên Internet

Gõ /deploy. AI hướng dẫn từng bước: chọn hosting, mua domain, cấu hình DNS, tự động setup SSL, SEO, Analytics, Privacy Policy...

User: /deploy
AI: "Anh có domain chưa? Muốn deploy lên Vercel hay VPS riêng?"
AI: "Đang build production... Đã setup SEO, Analytics. URL: https://myapp.com"
8
/save-brain Lưu kiến thức

Cuối buổi, gõ /save-brain. AI tự động update tất cả tài liệu (Architecture, API Docs, Changelog). Ngày mai bạn chỉ cần gõ /recap là nhớ lại tất cả.

Cài đặt trong 30 giây

Mở Terminal trong thư mục dự án và chạy 1 lệnh.

📁 Cài cho 1 dự án (Khuyến nghị)

🪟

Windows (PowerShell)

Nhấn Ctrl + ` để mở Terminal

iex (irm https://raw.githubusercontent.com/TUAN130294/awf/main/install.ps1)
🍎

Mac / Linux

Mở Terminal.app hoặc iTerm

curl -fsSL https://raw.githubusercontent.com/TUAN130294/awf/main/install.sh | sh

🌍 Cài Global (Cho tất cả dự án)

Cài 1 lần, tự động có AWF trong mọi dự án mới. Thêm flag -Global (Windows) hoặc --global (Mac):

🪟 Windows:

iex "& { $(irm https://raw.githubusercontent.com/TUAN130294/awf/main/install.ps1) } -Global"

🍎 Mac/Linux:

curl -fsSL https://raw.githubusercontent.com/TUAN130294/awf/main/install.sh | sh -s -- --global

⚠️ Lưu ý: Global install đặt workflows vào ~/.agent/workflows/. Antigravity sẽ tự động tìm thấy nếu support.

Sau khi cài xong, restart Antigravity/IDE và gõ / để thấy các lệnh mới.

Câu hỏi thường gặp

AWF hoạt động với IDE nào?

AWF được thiết kế cho Cursor và Windsurf - các IDE có tích hợp AI Agent. Nó cũng có thể hoạt động với GitHub Copilot Chat nếu bạn copy nội dung workflow vào context.

AWF có miễn phí không?

Hoàn toàn miễn phí và Open Source (MIT License). Tuy nhiên, bạn vẫn cần subscription của Cursor hoặc Windsurf để sử dụng AI Agent.

Tôi không biết code, có dùng được không?

Đó chính là mục tiêu của AWF! Bạn chỉ cần mô tả ý tưởng bằng tiếng Việt bình thường. AI sẽ tự lo phần kỹ thuật. Tuy nhiên, hiểu biết cơ bản về cách hoạt động của web sẽ giúp bạn giao tiếp với AI hiệu quả hơn.

Làm sao để cập nhật AWF?

Chạy lại lệnh cài đặt. Script sẽ tự động ghi đè các file workflow cũ bằng phiên bản mới nhất.
