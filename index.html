Cài đặt Global

Chỉ cần chạy 1 lần duy nhất. Có mặt ở mọi dự án.

🪟 Windows

Copy và dán vào PowerShell:

iex (irm https://raw.githubusercontent.com/TUAN130294/awf/main/install.ps1)

🍎 Mac / Linux

Copy và dán vào Terminal:

curl -fsSL https://raw.githubusercontent.com/TUAN130294/awf/main/install.sh | sh

Sau khi chạy xong, hãy Restart Antigravity/Cursor. Workflows sẽ tự động nhận diện ở mọi dự án.

FAQ

AWF có hoạt động với GitHub Copilot không? ↓

AWF được tối ưu cho Antigravity trong Cursor/Windsurf vì nó dùng file Markdown workflow. Tuy nhiên, bạn có thể copy nội dung file vào chat của Copilot, nó vẫn hiểu nhưng kém hiệu quả hơn.

Code được sinh ra có an toàn không? ↓

Có. Workflow /code của AWF được thiết kế để luôn nhắc AI kiểm tra các lỗ hổng bảo mật phổ biến (SQL Injection, XSS) trước khi viết code.
